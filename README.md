# bmi706-final-project

## Data visualization of CMS data

[View Streamlit App](https://bmi706-final-project-main-rq6kmxmiqpppggxhdty6dh.streamlit.app/)

We created a Streamlit App for visualizing trends around Medicare Spending per Beneficiary at the state and hospital level. Data sources below:

- [Medicare Spending Per Beneficiary by State](https://data.cms.gov/provider-data/dataset/rs6n-9qwg)
- [Hospital General Information](https://data.cms.gov/provider-data/dataset/xubh-q36u)
- [Medicare Spending Per Beneficiary Hospital](https://data.cms.gov/provider-data/dataset/rrqw-56er)
- [Healthcare Associated Infections - Hospital](https://data.cms.gov/provider-data/dataset/77hc-ibv8)
- [Payment and Value of Care Hospital](https://data.cms.gov/provider-data/dataset/c7us-v4mf)
- [Complications and Death Hospital](https://data.cms.gov/provider-data/dataset/ynj2-r877)
